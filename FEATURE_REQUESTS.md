# window-tool Feature Requests

Feature requests and ideas for the macOS CLI window management tool.

## Open

## Completed

### FR-1: `sidebar --side=top`

Add `--side=top` support to the sidebar command. Pins a window to the top edge of the screen. With `--full-width`, the window is maximized to the full screen width.

**Done in v0.16.0**

### FR-2: `sidebar --side=bottom`

Add `--side=bottom` support to the sidebar command. Pins a window to the bottom edge of the screen. With `--full-width`, the window is maximized to the full screen width.

**Done in v0.16.0**
