# window-tool Feature Requests

Feature requests and ideas for the macOS CLI window management tool.

## Open

### FR-1: `sidebar --side=top`

Add `--side=top` support to the sidebar command. Pins a window to the top edge of the screen. With `--full-height` (or a new `--full-width`), the window should be maximized to the full screen width.

### FR-2: `sidebar --side=bottom`

Add `--side=bottom` support to the sidebar command. Pins a window to the bottom edge of the screen. With `--full-height` (or a new `--full-width`), the window should be maximized to the full screen width.
